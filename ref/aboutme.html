ผมเป็นใคร

ผมมีชื่อเต็มว่า นาย จักรพรรดิ์ นิทรัพย์ กำลังเขียนโค้ดใน VS Code สลับไปมากับทดสอบโค้ดใน Brave จนตาลายไปหมด...

ผมเกิดวันที่ 12 กันยายน 2005

ชื่อเล่น รายู

งานอดิเรก เล่นเกมและทำมีมเกี่ยวกับคอมพิวเตอร์ในโลกอินเตอร์เน็ต และตอบปัญหาทางเทคนิคให้กับเหล่าผู้คนในพันทิป

ความสามารถพิเศษ การเขียนโค้ด ทักษะภาษาอังกฤษ ทักษะการพูดและการสื่อสาร และทักษะที่เกี่ยวกับเทคนิคคอมพิวเตอร์

สิ่งที่อยากเรียนรู้เพิ่ม ผมกำลังคิดที่จะเรียน Interactive Web อยู่ครับ นอกจากนี้ก็มี Framwork ต่างๆ หลายตัวที่ผมอยากทำความรู้จักให้มากขึ้น

สิ่งที่คาดหวังจากมหาวิทยาลัย ทักษะที่จะสามารถทำให้ผมเป็น Developer เต็มรูปแบบ แน่นอนว่า FullStack Developer นั้นเป็นเป้าหมายหลัก แต่ผมไม่คิดที่จะหยุดแค่นั้น มันจะเป็นไปได้ไหมที่ผมจะเรียนรู้ทักษะ Software Engineer จากมหาลัยนี้?
